## Project Description

Smart Finance Planner is an interactive personal budgeting web application that helps users manage income, expenses, savings goals, and category limits.

It provides visual insights like spending charts, heatmaps, risk profiling, gamified achievements, and progress tracking — all stored locally in the browser.

---

## The Problem Statement

Many students and young professionals:

* Do not track monthly spending
* Overspend without realizing
* Have no structured savings goals
* Lack visual insights into financial habits
* Struggle with maintaining budgeting consistency

Most budgeting apps are complex, paid, or overwhelming.

---

## The Solution

Smart Finance Planner provides:

* Simple monthly income & expense tracking
* Category-based spending limits
* Savings goal tracking
* Risk profile calculation
* Visual analytics (Chart + Heatmap)
* Gamified achievements & XP system
* Local storage (no login required)

It makes budgeting intuitive, motivating, and visually engaging.

## Diagrams

### System Architecture

![Architecture Diagram](docs/architecture.png)

**Architecture Overview:**

User → Browser UI → JavaScript Logic → LocalStorage
Chart.js renders analytics dynamically
All data persists using browser LocalStorage

No backend required.

---

### Application Workflow

![Workflow](docs/workflow.png)

1. User enters income & fixed costs
2. Adds expenses by category
3. Sets category limits
4. Savings auto-calculated
5. Risk profile auto-detected
6. Charts & heatmap update dynamically
7. Achievements & XP update based on savings ratio
